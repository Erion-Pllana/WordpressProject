# WordpressProject Theme

This is a simple WordPress theme scaffold with multiple page templates.

Activation:

1. Place the `WordpressProject` theme folder into `wp-content/themes/` (already present in this workspace).
2. In WordPress admin go to Appearance → Themes and activate "WordpressProject".

Files added:
- `style.css` – theme header
- `functions.php` – theme setup and asset enqueues
- `header.php`, `footer.php`, `sidebar.php` – layout
- Template files: `index.php`, `front-page.php`, `page.php`, `single.php`, `archive.php`, `404.php`
- Page templates: `page-about.php`, `page-contact.php` (contact form uses `wp_mail`)
- `comments.php` – standard comments
- `assets/css/main.css` and `assets/js/main.js`
